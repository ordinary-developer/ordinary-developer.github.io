Qt resources

It is said that a default GUI framework for C++ is Qt (Of course there are other GUI frameworks such as Gtk, wxWidgets etc., but Qt is a mainstream).

So to be a competitive c++ developer it is necessary to know the Qt framework.

Here some books devoted to the Qt framework are listed:

Qt 5.10 Professional Programming with C++ by Max Schlee

This book is available only in Russian. But it is a good introductory book. There are many other books but this one is, in my opinion, the most suitable for learning Qt from scratch

Other resources

Official qt docs contain an index of Qt books (in different languages) and can be very useful for further learning.
