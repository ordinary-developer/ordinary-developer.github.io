Online C++ compilers

Online C++ compilers

Very often it is necessary to test some simple code in c++. Online compilers can be used for such purposes.

Here I shall enumerate some noteworthy online compilers.

Base Intstruments

Wandbox.org

This is a good set of online compilers (not only c++) with different last version c++ compilers (gcc, clang, zapc). It also allows to use boost libraries and to edit a text in Emacs and Vim modes.

Codeinterview.io

This is instrument is mainly for remote interviews, but it allows to run code in a demo mode. Also, it allows editing a text in Emacs and Vim modes.

Coliru.stacked-crooked.com

This is a good simple online c++ compiler that supports the c++17 standard.

Rextester.com

This is a good simple set of compilers that also supports vc++ compiling and boost libraries.

Cpp.sh

This is also a good simple c++ compiler. Unfortunately, it supports only the c++14 standard.

More exotic instruments

Onlinegdb.com

This is a set of online compilers with a debug opportunity.

Godbolt.org

This is a set of instruments that allows showing the assembly output of the compiled code.

Quick-bench.com

This is an instrument for benchmarking c++ code (also can show the assembly output of code).
