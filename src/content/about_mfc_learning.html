About MFC learning

Despite now Microsoft MFC is an obsolete technology there are many legacy projects that use it. And this post contains some materials for learning MFC.

Books

Programming Windows with MFC (2-nd ed.) by Jeff Prosise

It is a good introductory book about MFC. It contains many examples and most importantly first examples are written from scratch to show the all internal cookery of MFC.

MFC Programming from the Ground Up by Herbert Schildt

It is also a good introductory book about MFC.

Programming Microsoft Visual C++ (5-th ed.) by David J.Kruglinski et al.

It is a more comprehensive book but it assumes that a reader has no MFC knowledge. So, in theory, it also can be recommended for beginners.

MFC Programming with Visual C++ 6 Unleashed by David White et al.

It is also a comprehensive book and also assumes that a reader has no MFC knowledge.

The two first books may be regarded as an absolute minimum for MFC and the other two books can be regarded as additional materials.

Development Environment

And now about a development environment for MFC.

Of course, modern versions of Microsoft Visual Studio support creating MFC applications, but these versions are bulky and heavy. It is better to use a virtual machine with Windows XP and Microsoft Visual Studio 2003 for learning MFC. But it is of course only my opinion.
